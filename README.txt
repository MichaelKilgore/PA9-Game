This is the first thing that should be read before trying to clone this repository, also should be rough guidelines to be followed while coding/committing

Github desktop is a pretty good git client, gitKraken is also good, recommend to use either for managing the repository

When working on something try to make the variable names useful and comment at least a little, especially when doing something confusing/new

Use seperate branches for different aspects of the game (EX -> one branch for the player object, one branch for the game logic, etc.)
